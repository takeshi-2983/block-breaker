# ブロック崩し

シンプルで楽しいブロック崩しゲームです。ブラウザで直接プレイできます。

## ゲーム説明

パドルを操作してボールをブロックに当て、すべてのブロックを破壊してクリアを目指すゲームです。

### 操作方法

- **マウス移動**: パドルを左右に動かす
- **ゲーム開始**: 「ゲーム開始」ボタンをクリック
- **一時停止**: 「一時停止」ボタンをクリック
- **リセット**: 「リセット」ボタンをクリック

### ゲームルール

- ボールがパドルに当たると反射します
- ボールがブロックに当たるとブロックが破壊され、スコアが加算されます
- ボールが画面下に落ちるとライフが1減ります
- ライフが0になるとゲームオーバーです
- すべてのブロックを破壊するとクリアです

### スコア

- ブロック1個破壊: 10ポイント
- 最大スコア: 320ポイント（32個のブロック）

## ファイル構成

- `index.html`: ゲームのHTMLファイル
- `style.css`: ゲームのスタイルシート
- `game.js`: ゲームのロジック

## 使い方

1. `index.html`をブラウザで開く
2. 「ゲーム開始」ボタンをクリック
3. マウスを動かしてパドルを操作
4. ゲームを楽しむ！

## 技術仕様

- **言語**: HTML5, CSS3, JavaScript (Vanilla)
- **グラフィックス**: Canvas API
- **ブラウザ対応**: モダンブラウザ全般
